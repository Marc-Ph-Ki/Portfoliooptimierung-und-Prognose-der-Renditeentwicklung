# Portfoliooptimierung-und-Prognose-der-Renditeentwicklung
In diesem Projekt wird ein optimales Aktienportfolio unter Berücksichtigung von Rendite und Risiko entwickelt und evaluiert. Ausgangspunkt bildet die Problemstellung, wie Anleger ihre Kapitalallokation so gestalten können, dass die Rendite maximiert und gleichzeitig das Risiko minimiert wird. Nach der Datenaufbereitung werden zentrale Risikokennzahlen wie diskrete und stetige Renditen sowie der Value-at-Risk berechnet. Anschließend erfolgt eine explorative Datenanalyse zur Identifikation von Mustern, Korrelationen und potenziellen Diversifikationseffekten zwischen den Assets. Mithilfe eines Monte-Carlo-Ansatzes werden zahlreiche Portfoliozusammensetzungen simuliert, um die effiziente Grenze zu approximieren und das Portfolio mit der höchsten Sharpe Ratio zu bestimmen. Die Modellevaluation umfasst Performance- und Risikokennzahlen sowie Szenarioanalysen. Abschließend werden die Ergebnisse interpretiert, um fundierte Rückschlüsse für eine rationale Portfolioentscheidung zu ermöglichen.
